# QBCore FistControl Resource 🥊

## What is QB-FistControl?

QB-FistControl is a **high-performance FiveM resource** designed for **QBCore servers** that prevents accidental fistfights and implements an intelligent **Right-Click + Left-Click combo system** for controlled punching mechanics.

## Features at a Glance

✅ Prevents accidental fistfights  
✅ Right-Click + Left-Click combo system  
✅ Disables R key for punching  
✅ Removes normal left-click attacks  
✅ Smooth animations  
✅ Fully configurable  
✅ Zero performance impact  
✅ QBCore native integration  

## Installation (Quick)

```bash
# Download
git clone https://github.com/amirragzon3/qb-fistcontrol.git

# Install - Place in resources/[qb]/qb-fistcontrol/
# Add to server.cfg: ensure qb-fistcontrol
# Restart server
```

## How to Use

1. **Hold Right-Click** (Mouse 2)
2. **Press Left-Click** (Mouse 1) while holding
3. **Punch animation plays** ✓
