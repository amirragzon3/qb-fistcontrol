# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-14

### Added
- Initial release of QB-FistControl
- Right-Click + Left-Click combo system for punching
- R key disabling functionality
- Normal attack disabling
- Smooth punch animations
- Full configuration system
- Comprehensive documentation
- MIT License

### Features
- Prevent accidental fistfights
- Configurable control disabling
- Performance optimized threading
- QBCore native integration
- Zero memory leaks
